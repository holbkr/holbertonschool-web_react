# 0. Set TailwindCSS

Integrate TailwindCSS v4 into the `dashboard` React project without using any TailwindCSS configuration file, and configure the Roboto font (regular, medium and bold weights) as the default font for the application.

## Files

- `dashboard/src/main.css`: TailwindCSS import, theme configuration for the Roboto font family
- `dashboard/src/main.jsx`: imports the Roboto font weights (400, 500, 700) from `@fontsource/roboto`
