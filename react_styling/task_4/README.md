# 4. Go Responsive

Make the Holberton Dashboard application responsive using TailwindCSS breakpoints, mobile-first.

## Files

- `dashboard/src/Notifications/Notifications.jsx`: full-screen drawer below 912px
- `dashboard/src/Notifications/NotificationItem.jsx`: responsive text sizing, border and padding
- `dashboard/src/App/App.jsx`: responsive container and flexbox layout
- `dashboard/src/Header/Header.jsx`: responsive layout and text sizes below 520px
- `dashboard/src/Login/Login.jsx`: responsive form sizing
- `dashboard/src/CourseList/CourseList.jsx`: responsive table container
- `dashboard/src/Footer/Footer.jsx`: responsive text sizing, stuck to the bottom
